API HTML 5

  • The restriction of using only a string value can be overcome by encoding any JavaScript object or value as a JSON string and then performing a type conversion later as needed.
  • Data attributes provide a convenient way to add data directly into HTML markup, allowing for a richer user experience.
  • The fact that only strings can be saved may seem like a restriction at first, but by using JSON, we can store any JavaScript object in local storage
  • The Web Storage API allows key-value pairs to be stored on the user's device in a similar way to cookies, but without the same storage restrictions.
