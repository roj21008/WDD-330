Transforms and Transitions

  • The CSS3 transform property allows you to translate, rotate, scale and/or skew any element on the page.
  • We can manipulate the appearance of an element using transformation functions . The value of the transform property is one or more transform functions (separated by spaces) that will be applied in the order they are supplied
  • The translation functions allow you to move elements left, right, up, or down.
  • As with translate , you can also use the scaleX (x) or scaleY (y) functions . These functions will scale only the horizontal dimensions or only the vertical dimensions, respectively.
  • To declare multiple transformations, provide a space-separated list of transform functions.

Canvas, SVG, and Drag and Drop

  • XML stands for Extensible Markup Language. Like HTML, it is a markup metalanguage. In plain language, it is a system for annotating text.
  • Unlike the canvas, images created with SVG are available through the DOM.
  • To view an SVG file, simply open it via the File menu in any browser that supports SVG
  • Both canvas and SVG allow you to draw custom shapes, paths, and fonts.
